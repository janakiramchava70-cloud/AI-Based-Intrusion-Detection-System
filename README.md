# AI-Based-Intrusion-Detection-System# AI-Based Intrusion Detection System

## Overview
This project presents an AI-based Intrusion Detection System (IDS) developed using deep learning techniques. The system is designed to detect malicious network activities and classify anomalies by leveraging Autoencoders and a Multilayer Perceptron (MLP), improving the accuracy of intrusion detection.

## Features
- Detects anomalies in network traffic
- Uses Autoencoders for feature learning
- Classifies intrusions using a Multilayer Perceptron (MLP)
- Improves network security through intelligent threat detection

## Technologies Used
- Python
- TensorFlow
- Scikit-learn
- NumPy
- Pandas
- Autoencoders
- Multilayer Perceptron (MLP)

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Extraction
4. Autoencoder Model Training
5. Intrusion Classification using MLP
6. Performance Evaluation
